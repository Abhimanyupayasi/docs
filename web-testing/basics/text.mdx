# Text Locator

The Text locator targets elements whose visible text matches a value. Itâ€™s helpful for buttons, links, and labels.

```html
<button>Continue</button>
```

Example usage

- Target: text=Continue
- Behavior: Selects a clickable element whose text equals "Continue".

<Caution>
Text can change due to localization or content updates. Prefer attributes when possible.
</Caution>
