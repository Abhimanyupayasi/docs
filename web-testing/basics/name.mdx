# Name Locator

The Name locator selects elements by their name attribute, commonly used in forms.

```html
<input name="username" />
```

Example usage

- Target: name=username
- Behavior: Selects the form control whose name is "username".

<Tip>
Name is useful when IDs are dynamic but form control names are stable.
</Tip>
