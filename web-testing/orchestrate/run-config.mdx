# Run Configuration in Test Suite

In **NextGen QE**, you can manage suite-level execution settings directly from the **Orchestrate** tab.  
This allows you to define how your test suite runs â€” including browser type, environment, and execution mode.

---

## Steps to Manage Run Configuration

### Step 1: Open the Orchestrate Tab
Navigate to the **Orchestrate** tab from the main navigation menu.

---

### Step 2: Select the Test Suite
Choose the **Test Suite** for which you want to configure or update the run settings.

---

### Step 3: Click on Run Config
Select **Run Config** to open the suite-level configuration window.

<img src="/web-testing/imgs/run-conf-1.png" alt="Suite run configuration in NextGen QE" />

---

### Step 4: Update Suite Configuration
Here, you can update suite-specific settings such as:
- Browser type  
- Execution environment  
- Parallel or sequential run mode  

Once updated, save your configuration.
<img src="/web-testing/imgs/run-conf-2.png" alt="Suite run configuration in NextGen QE" />
