## Overview

The **Page Objects** tab is where all your test pages and their elements (objects) are stored.  
You can define them using **XPath**, **CSS class**, **ID**, or other HTML selectors.

---

## Steps to Create Page Objects

### 1. Go to PiedaP Platform
Visit [https://platform.piedap.io](https://platform.piedap.io) and log in to your account.

### 3. Click on the Objects Tab  
Select the **Objects** tab to manage or create new page objects.

### 4. Create a Page

### 5. Open the Object Icon  
After creating the page, click on the **Object** icon to open the object management area.  
![Open Object Management Section in NexGen QE Page Repository](/web-testing/imgs/ObjectsClick.png)

---

### 6. Create a New Object  
Click **New Object** and fill in the following fields to define your element:  

- **Object Name***  
- **Id**  
- **Tag**  
- **Class Name**  
- **Name**  
- **Text**  
- **Xpath***  
- **Description**
