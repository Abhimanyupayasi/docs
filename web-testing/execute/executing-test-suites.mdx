# Executing Test Suites in NextGen QE

After creating a **Test Suite** in the **Orchestrate** tab, you can execute it from the **Execute** tab.  
This section allows you to run, monitor, and manage your test executions in real time.

---

## Steps to Execute a Test Suite

### Step 1: Navigate to the Execute Tab

Go to the **Execute** tab from the main navigation panel.

<img src="/web-testing/imgs/execute_tab_1.png" alt="Navigate to the Execute tab in NextGen QE" />

---

### Step 2: Select the Test Suite

From the list of available test suites, select the one you want to execute.

<img src="/web-testing/imgs/execute_tab_2.png" alt="Select a test suite to execute" />

---

### Step 3: Start Execution

Click the **Execute** button to begin running the selected test suite.

<img src="/web-testing/imgs/execute-click.png" alt="Start executing a test suite" />

---

## Test Suite Execution

Once the execution starts, the system will process your test cases based on the selected configuration and strategy.  
You can track progress, view detailed logs, and analyze the results after completion.

---

**Result:**  
Your test suite has been executed successfully.  
To learn more about monitoring and analyzing results, refer to [Viewing Execution Reports](/web-testing/execute/view-reports).
