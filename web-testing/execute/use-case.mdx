# Executing a Use Case

Once the **Test Suite** has been created under a use case in the **Orchestrate** tab, you can execute it directly from the **Execute** module.  
This ensures the end-to-end validation of your defined testing workflow.

---

## Steps to Execute a Use Case

### Step 1: Go to the Execute Tab
Open the **Execute** tab from the main navigation menu.

---

### Step 2: Navigate to the Use Case
Select the **Use Case** section under the Execute module.
